# Yet Another Proxy
A HTTP proxy based on the `Host` header (eg: `plex.example.org` > `localhost:32400`)

## Features
* Cookie authentication with page
* Basic authentication
* Authentication bypass depending on IP/IP Range
* Disable robots
* Display real IP
* Whitelist
* Blacklist
* Redirections
* Easy to configure
